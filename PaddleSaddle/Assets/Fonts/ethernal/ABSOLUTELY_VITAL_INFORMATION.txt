Ethernal is a two-version script typeface. 
It has a beautiful flow, perfected curves and a classic style. 
Created by Måns Grebäck, this decorative typeface also has a lot of 
contextual alternates, as well as stylistic alternates and ligatures, 
to make it as flexible as possible. 
The fonts supports a very wide range of languages. 

Use ( ) [ ] { } after a word for different endings. 
Example: Ethernal[
